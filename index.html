Colors

The first thing we will do is change the colors of the headers to our favorite color. 

h2 {
	color: #14305e;
}

We can also add a background color to elements. I will make the background of code snippets to be grey, as such: 

pre {
	background-color: #eeeeee
}

Fonts

We can also choose specific fonts for specific content. 

body {
	font-family: Helvetica, sans-serif;
}
pre {
	font-family: 'Courier New', monospace;
}

Box model (padding, margin, border, outline)

Div's represent blocks of content. We can add spacing around them in many different ways, such as padding and margin. We can also identify them with a border or an outline. The difference between these is shown in the box model diagram below. 

#boxmodel {
	display: block;

	padding: 2em;
	margin: auto;
	border: 1px solid black;
}
header, section, pre {
	padding: 1em;
}

Max-width

A common way to make a website more readable these days is to limit the max width of a website. This makes it easier to read. We can do that as such: 

main {
	margin: auto;
}
main {
	max-width: 50em;
}

Block vs inline elements

Block elements start on new lines, and take up the full width. Inline elements take up only as much width as necessary. Many inline elements can be placed beside one another.

HTML Div elements are block elements. HTML Span elements are inline elements by default.

We can also specify the behavior that we want using CSS 'display' property as such:

Example of
divs with 'display inline' 
.inline-divs {
	display: inline;
}

Position

We can also specify the position of an element on the page. We can use position, in conjunction with top and left to specify the position of an element. This allows us to have interesting designs for our page.

A common example is to make the header stickied to the top of the page. We can do that as such: 

header {
	position: sticky;
	top: 0;
	background-color: white;
	border-bottom: 1px solid black;
}

We could also place an image to the right within a div by doing so: 

#smiley {
	position: relative;
	left: 400px;
}

Note that the smiley is now 'over' the header when we scroll, which is odd. We'll adjust this by making the 'depth', otherwise know as the 'z-index' of the header to 1. This will put the header 'on top' of other elements. 

nav {
	z-index: 1;
}

Hover

We can add specific styling upon hover only. I want my links to be light-green when hovered over. We can do that by specifying the tag, along with the :hover pseudo-class. 

a:hover {
	color: light-green;
}
